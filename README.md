## About This Workshop

This is a **complete, self-paced C programming workshop** designed specifically for college students with **zero prior C knowledge**. Whether you're a complete beginner or coming from another programming language, this guide will take you from basic syntax to advanced data structures and algorithms.

### Why This Guide?

- **Beginner-Friendly**: Assumes zero C knowledge
- **Comprehensive**: Covers basics to advanced topics
- **Practical**: 145+ practice problems with solutions
- **Visual**: ASCII diagrams, flowcharts, and memory visualizations
- **Interview-Ready**: Includes DSA and coding challenges
- **Self-Paced**: Learn at your own speed

## Getting Started

### Step 1: Set Up Your Environment

**Install GCC Compiler:**

**Windows:**
```bash
# Download MinGW or install via chocolatey
choco install mingw
```

**Mac:**
```bash
# Install Xcode Command Line Tools
xcode-select --install
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install build-essential
```

**Verify Installation:**
```bash
gcc --version
```

### Step 2: Set Up Your Editor

**Recommended: Visual Studio Code**
1. Download from [code.visualstudio.com](https://code.visualstudio.com)
2. Install C/C++ extension by Microsoft
3. Configure for C development

### Step 3: Start Learning

```bash
# Clone this repository
git clone https://github.com/cy-infamous/C-language-guide.git
cd C-language-guide

# Start with basics
cd 01-Basics
# Open 01-introduction.md and start reading!
```

### Step 4: Write Your First Program

Create `hello.c`:
```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

Compile and run:
```bash
gcc hello.c -o hello
./hello              # Linux/Mac
hello.exe            # Windows
```
